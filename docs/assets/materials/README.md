# Course Materials Bundle

This folder contains the local, course-organized copy of the teaching material used in the SoSe 2026 block seminar.

## Structure

- `day1/`: setup, config basics, island introduction material and configs.
- `day2/`: output analysis, landscape building material, and Day 2 datasets.
- `day3/`: sandbox, critical reflection notes, advanced configs, and scripts.
- `shared/integrated_core/`: cross-day integrated workshop files and configs.
